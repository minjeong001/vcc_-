# 🛒 시각장애인을 위한 탈부착형 쇼핑카트 보조 모듈

## 📝 프로젝트 개요
쇼핑은 일상에서 빠질 수 없는 활동이지만, 시각장애인에게는 여전히 많은 제약이 따릅니다. 본 프로젝트는 **기존 쇼핑카트에 탈부착 가능한 보조 모듈**을 제공함으로써, 시각장애인이 타인의 도움 없이 **독립적으로 쇼핑을 수행**할 수 있도록 지원합니다.

## 🎯 주요 기능

1. ### 🏷️ 상품 및 유통기한 식별
   - **YOLOv8 모델**을 통해 상품을 인식합니다.  
   - **OCR 기술(Pytesseract)**을 활용해 유통기한을 판독하고, **gTTS 음성 안내**로 사용자에게 정보를 제공합니다.

2. ### 📢 상세 정보 음성 안내
   - 상품 인식 후 **SQLite 데이터베이스**에서 가격, 칼로리, 영양 성분 등 정보를 조회하여 **음성으로 출력**합니다.

3. ### 🚧 실시간 장애물 감지
   - **아두이노 + 초음파 센서**를 활용해 전방 장애물 거리 측정 및 사용자에게 **실시간 경고**.

4. ### 🧭 실내 경로 안내
   - 매장 내 위치 추적 기능으로 **원하는 상품 코너까지의 경로 안내** 지원 (기본적 내비게이션 알고리즘 또는 Beacon 기반 가능).

## 🛠 기술 스택

| 분류 | 기술 구성 |
|------|------------|
| **하드웨어** | Raspberry Pi, Arduino, 초음파 센서, PiCamera2 |
| **소프트웨어** | Python, C++, OpenCV, YOLOv8, Pytesseract, gTTS, SQLite |

## 📂 파일 구조

- `main.py` : 전체 시스템 제어 및 버튼 입력 이벤트 처리  
- `product.py` : YOLO 기반 상품 인식 처리  
- `expiry.py` : Pytesseract로 유통기한 OCR 인식  
- `voice.py` : gTTS를 통한 음성 출력 및 DB 정보 연동

## 🚀 기대 효과

- **시각장애인의 자율적인 쇼핑 환경 제공**  
- **충돌 방지를 통한 안전성 확보**  
- **사회적 약자 배려 기술로서의 확장성**  
- **다양한 유통 환경에 적용 가능한 범용 모듈**
